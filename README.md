# GEO-TREES tree inventory quality control toolbox

This repository contains documentation and code for quality assurance (QA) and quality control (QC) of GEO-TREES tree inventory data before it is processed in the [GEO-TREES PDA_processing workflow](https://github.com/GEO-TREES/PDA_processing) to generate estimates of above-ground woody biomass density (AGBD).

This QA-QC workflow relies on both the [TreeData](https://github.com/VincyaneBadouard/TreeData) R package and the [ATFS DataHarmonization Shiny App](https://github.com/Alliance-for-Tropical-Forest-Science/DataHarmonization).
